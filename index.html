Bands in Vans

  • Technologies Used

    I really wanted to make sure the website was responsive, as bands or DIY music fans are usually closer to a phone than a computer screen. I used C#/.NET, Razor, ASP.NET MVC, HTML/CSS, and Bootstrap for this project.

    This project grew from my love of DIY rock and roll. My goal for this project was to make a resource similar to DoDIY, or the now defunct Book Your Own Life, but with more of a review spin. I had a lot of fun with getting the glyphicons of stars to populate correctly. Razor logic and picture of result below.

Bank Account

  • Technologies Used

    This project is a C# console application created using the object-oriented programming principles of inheritance and polymorphism. I paid special attention layout on the console view and setting up classes in a logical and realistic way.

  • Problems and Solutions

    I had a lot of fun playing with parameters for the different methods! Passing in an object and using its properties to manipulate it or perform operations was instructive. I can see how more complex programs in industry would rely upon careful design of parameters and properties, especially where protecting user information is concerned.

Tuscany Villa

  • Technologies Used

    I used HTML, CSS, the Bootstrap library, and a little bit of JavaScript. I like to think of the website like a building: HTML is the structure, CSS is the styling, Bootstrap is there to make it accessible to everyone, and JavaScript is the motion and brightness.

  • Problems and Solutions

    The jumbotron up above gave me the biggest problem with the page. I had a time trying to get the map image on the right trying to line up with the image of the villa on the right. My solution was to make the villa picture the background image for the entire jumbotron class. Then, I made a column for the text on the left, and a column for the image on the right. Check out my code below. Some styling, and voila!

  • GitHub

    Styling, styling, styling. Here, version control made it easy for me to think like a drafter. I could try an entirely different style, change my entire style sheet, and not lose my work if I wanted to crumple it up and throw it away. More than once, I changed something, and then changed it back. The creative process, am I right?
